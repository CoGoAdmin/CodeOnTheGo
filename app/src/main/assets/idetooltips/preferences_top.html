The following categories of preferences are available.

Build and Run Settings for Gradle and app behavior after installation. Learn more about Build and Run settings.
Termux Settings for the Termux terminal. Learn more about Termux settings.
GitHub Preferences for GitHub interaction. Learn more about GitHub settings.
Privacy Enable or disable anonymous collection of usage statistics. Do we even use this?
Developer options Enable or disable log dumping and showing application logs in the IDE.
